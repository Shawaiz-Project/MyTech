## üìÑ Managing Website Content

### 1. **Services Management**
   
   **What it does:** Manage the services you offer to clients
   
   **How to add a service:**
   - Click "Services" ‚Üí "Add New Service"
   - Fill in:
     - **Service Title** - Name of your service
     - **Short Description** - 1-2 line summary
     - **Detailed Description** - Full details about the service
     - **Technologies Used** - Tools/tech stack for this service
     - **Icon/Image** - Upload service icon (PNG, JPG, GIF)
     - **Status** - Active or Inactive
   - Click "Save"
   
   **Edit a service:**
   - Click "Services" ‚Üí Find your service ‚Üí "Edit"
   - Make changes ‚Üí Click "Update"
   
   **Delete a service:**
   - Click "Services" ‚Üí Find your service ‚Üí "Delete"

### 2. **Projects/Portfolio Management**
   
   **What it does:** Showcase your completed projects and case studies
   
   **How to add a project:**
   - Click "Projects" ‚Üí "Add New Project"
   - Fill in:
     - **Project Title** - Name of the project
     - **Client Name** - Company/client name
     - **Category** - Type of project (Web, Mobile, Design, etc.)
     - **Description** - Detailed project overview
     - **Key Features** - Main features/achievements
     - **Technologies** - Tech stack used
     - **Project Link** - URL to live project (if available)
     - **Images** - Upload multiple project screenshots
     - **Status** - Active or Inactive
   - Click "Save"

### 3. **Blog Management**
   
   **What it does:** Share articles, news, and insights with your audience
   
   **How to add a blog post:**
   - Click "Blogs" ‚Üí "Add New Blog"
   - Fill in:
     - **Title** - Blog post title
     - **Category** - Blog category
     - **Content** - Main blog text
     - **Cover Image** - Main image for the post
     - **Tags** - Keywords for the post
     - **Status** - Draft or Published
   - Click "Save"
   
   **Note:** Only published blogs appear on the public website

### 4. **Team Management**
   
   **What it does:** Showcase your team members
   
   **How to add a team member:**
   - Click "Team" ‚Üí "Add New Member"
   - Fill in:
     - **Name** - Team member's full name
     - **Position/Title** - Job title
     - **Bio/Description** - Brief bio
     - **Profile Image** - Photo of team member
     - **Email** - Contact email
     - **Phone** - Phone number
     - **Social Links** - LinkedIn, Twitter, etc.
   - Click "Save"

### 5. **Job Postings**
   
   **What it does:** Post open job positions and manage applications
   
   **How to add a job:**
   - Click "Jobs" ‚Üí "Add New Job"
   - Fill in:
     - **Job Title** - Position name
     - **Department** - Department name
     - **Description** - Full job description
     - **Requirements** - Required skills/experience
     - **Salary Range** - Optional: salary info
     - **Job Type** - Full-time, Part-time, Contract
   - Click "Save"
   
   **View Applications:**
   - Click "Jobs" ‚Üí "View Applications"
   - See all job applications with candidate details
   - Download resume or contact the applicant

### 6. **Technologies Stack**
   
   **What it does:** Display the technologies/tools you use
   
   **How to add a technology:**
   - Click "Technologies" ‚Üí "Add New Tech"
   - Fill in:
     - **Tech Name** - Name of technology
     - **Category** - Backend, Frontend, Database, etc.
     - **Logo/Icon** - Technology icon
   - Click "Save"

### 7. **Testimonials**
   
   **What it does:** Display client feedback and reviews
   
   **How to add a testimonial:**
   - Click "Testimonials" ‚Üí "Add New"
   - Fill in:
     - **Client Name** - Who gave the testimonial
     - **Company** - Client's company
     - **Message** - The testimonial text
     - **Rating** - Star rating (1-5)
     - **Image** - Client photo (optional)
   - Click "Save"

### 8. **FAQ Management**
   
   **What it does:** Answer common customer questions
   
   **How to add FAQ:**
   - Click "FAQs" ‚Üí "Add New FAQ"

### 9. **Contact Messages**

## üìã Troubleshooting

### Problem: Can't login to admin panel

**Solution:**
- Check your username and password (default: Shawaiz / Shawaiz231980079)
- Make sure Caps Lock is off
- Clear browser cache and try again
- Use a different browser

### Problem: File upload not working

**Solution:**
- Check file type (PNG, JPG, GIF, WebP for images)
- Check file size (not too large)
- Make sure you have write permissions
- Try uploading a different file

### Problem: Changes not showing on website

**Solution:**
- Clear your browser cache (Ctrl+Shift+Delete)
- Hard refresh the page (Ctrl+F5)
- Wait a few seconds for the page to load
- Publish your changes if required

### Problem: WhatsApp button not appearing

**Solution:**
- Check if WhatsApp button is enabled in Site Settings
- Check if phone number is correct (with country code)
- Clear browser cache
- Try a different browser
